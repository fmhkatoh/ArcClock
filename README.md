# ArcClock

## What is this?

This is a [Rainmeter](https://www.rainmeter.net) skin.

Analog clock with calendar and systemmonitors

## Skin Description

This Rainmeter Skin package contains following skins.
 * Wather Forecast (1Week)
 * Weather map in Japan (JWA and Tokyo Amesh)
 * CPU Monitor
 * RAM Monitor

---

## これは何?

[Rainmeter](https://www.rainmeter.net) 用のスキンです。

アナログ時計に円弧でカレンダーとシステムモニターをつけました。

## スキンの説明

このRainmeterスキンには時計のスキン以外に以下のスキンを同梱しています。
 * 一週間分の天気予報
 * 実況天気図(日本気象協会(JWA)と東京アメッシュの地図を表示します)
 * CPU使用率のモニター
 * RAM使用率のモニター
